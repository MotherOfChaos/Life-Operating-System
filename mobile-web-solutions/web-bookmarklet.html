üîß Troubleshooting

‚ùå "Could not find chat input"

  • Make sure you're on claude.ai/chat (not homepage or docs)
  • Make sure the chat input box is visible on screen
  • Try refreshing the page and clicking the bookmark again

‚ùå "Failed to fetch" or 401 error

  • Your GitHub PAT might have expired
  • Check the PAT in the bookmarklet code
  • Regenerate PAT from GitHub settings if needed

‚ùå Nothing happens when I click the bookmark

  • Some browsers block JavaScript in bookmarks for security
  • Try a different browser (Chrome usually works best)
  • Check browser console (F12) for error messages

‚ö†Ô∏è Bookmarklet disappeared

  • You might have accidentally moved it
  • Just drag the button to your bookmarks bar again

üîê Security Notes

‚ö†Ô∏è Your GitHub PAT is visible in the bookmarklet code.

This is generally safe because:

  • Bookmarks are stored locally in your browser
  • PAT only has read access to one private repo
  • You can revoke it anytime from GitHub settings

To be extra cautious:

  • Don't export/share your bookmarks
  • Regenerate PAT periodically (every 3-6 months)
  • Use a browser you trust and keep updated
