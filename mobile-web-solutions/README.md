## ğŸ” SECURITY NOTES

### Personal Access Token (PAT)

Your GitHub PAT appears in:
- iOS Shortcut code (stored locally on device)
- Bookmarklet code (stored locally in browser)
- Smart Template text (in Notes/Keep app)
- Claude chat history (if using Smart Template)

**This is generally safe because:**
- PAT only has read access to one private repo
- Your devices are password-protected
- You can revoke/regenerate anytime
- Local storage is more secure than cloud

**Best practices:**
- Don't share these tools with others
- Don't post screenshots showing PAT
- Regenerate PAT every 3-6 months
- Use strong device passwords
- Log out of Claude on shared devices

**To regenerate PAT:**
1. GitHub â†’ Settings â†’ Developer settings â†’ Personal access tokens
2. Revoke old token
3. Generate new token (needs `repo` scope)
4. Update Smart Template, iOS Shortcut, and Bookmarklet with new token

## ğŸ› TROUBLESHOOTING

### Smart Template Issues

**"I can't fetch from GitHub"**
- Claude client may not have web fetch capabilities
- Use iOS Shortcut or Bookmarklet instead (they fetch locally)

**"403 Forbidden" or "401 Unauthorized"**
- PAT is invalid or expired
- Regenerate PAT from GitHub settings
- Update template with new PAT

### iOS Shortcut Issues

**"Untrusted Shortcut" warning**
- This is normal for shortcuts with web requests
- Settings â†’ Shortcuts â†’ Allow Running Scripts

**"Could not connect to GitHub"**
- Check internet connection
- Verify PAT is correct (no spaces)
- Test URLs in browser

**Shortcut doesn't run automatically**
- Check Automation settings
- Make sure "Ask Before Running" is OFF
- Check automation trigger conditions

### Bookmarklet Issues

**Nothing happens when I click**
- Make sure you're on claude.ai/chat
- Check browser console (F12) for errors
- Try a different browser (Chrome works best)

**"Could not find chat input"**
- Make sure chat input box is visible
- Refresh the page and try again
- Check if Claude UI has changed (bookmarklet may need update)

### General Issues

**Files are outdated**
- These tools fetch from GitHub `main` branch
- Make sure your CURRENT files are pushed to main
- Check GitHub web interface to verify files are latest

**Context is too large**
- Total files should be under ~50KB
- If larger, may need to split into multiple messages
- Or ask Claude to fetch files one at a time
